# css-unit-translator

Translates between CSS Units

## Installation

```js
npm install css-unit-translator
```

## Usage

```js
import CSSUnitTranslator from 'css-unit-translator'

let translator = New CSSUnitTranslator()

translator.translate(10, 'rem', 'px')
```
